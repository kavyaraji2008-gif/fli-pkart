# Flipkart Clone

A simple Flipkart clone built using HTML, CSS, and JavaScript.

## Features
- Home page
- Navigation bar
- Search bar
- Product cards
- Add to Cart button
- Responsive layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
Flipkart-Clone/
│── index.html
│── style.css
│── script.js
└── images/

## How to Run
1. Download the project.
2. Open the project folder.
3. Double-click `index.html`.
4. The website will open in your browser.
